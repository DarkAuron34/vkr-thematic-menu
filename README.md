# vkr-thematic-menu
Этот проект представляет собой веб-приложение, разработанное для автоматизации процесса составления тематических меню (например, для праздников, свадеб, дней рождения) и поиска подходящих ресторанов на основе сложного многокритериального запроса пользователя.  Проект разработан в рамках Выпускной квалификационной работы (ВКР) бакалавра в 2022 году.
